# blog

this is the source of my blog website

## important locations for stuff

|**what**              | **location**       |
|----------------------|--------------------|
|**home page content** | `posts/index.md`   |
|**posts**             | `posts/*.md`       |
|**pinned posts**      | `pages/index.tsx`  |
|**images**            | `public/img/*`     |
|**rss feed info**     | `rss/base.xml`     |
|**theme stuff**       | `styles/theme.css` |
